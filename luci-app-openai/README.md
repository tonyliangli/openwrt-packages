## How to build

- Enter in your openwrt dir

- Openwrt official SnapShots

  ```shell
  git clone https://github.com/sbwml/luci-app-openai package/openai
  make menuconfig # choose LUCI -> Applications -> luci-app-openai
  make V=s
  ```
